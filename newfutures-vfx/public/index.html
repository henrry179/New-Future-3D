安装指南

最后更新: 2024-12-19 阅读时间: 5 分钟

系统要求

操作系统

  • Windows 10+ (64位)
  • macOS 10.15+
  • Linux (Ubuntu 18.04+)

硬件要求

  • CPU: Intel i5 / AMD Ryzen 5 或更高
  • GPU: 支持 WebGL 2.0 的显卡
  • 内存: 8GB RAM (推荐 16GB)
  • 存储: 2GB 可用空间

浏览器支持

  • Chrome 90+
  • Firefox 88+
  • Safari 14+
  • Edge 90+

安装步骤

1

克隆仓库

git clone https://github.com/henrry179/New-Future-3D.git
cd New-Future-3D
2

安装依赖

cd newfutures-vfx
pip install -r requirements.txt
3

配置环境

cp config.env.example config.env
# 编辑 config.env 文件设置必要参数
4

启动服务

python start_server.py
提示: 首次启动可能需要几分钟来下载和初始化必要的资源。

验证安装

安装完成后，打开浏览器访问 http://localhost:8000，您应该能看到 NewFutures VFX 的主界面。

故障排除

端口被占用怎么办？

您可以通过修改配置文件或使用命令行参数指定不同的端口：

python start_server.py --port 8080
GPU 加速不工作？

确保您的浏览器支持 WebGL 2.0，并且显卡驱动程序是最新的。您可以访问 WebGL 2.0 测试页面 进行检查。

依赖安装失败？

请确保您使用的是 Python 3.8 或更高版本，并考虑使用虚拟环境：

python -m venv vfx-env
source vfx-env/bin/activate  # Linux/Mac
vfx-env\Scripts\activate     # Windows
pip install -r requirements.txt
